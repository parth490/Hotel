Introduction

Welcome to the Hotel Booking Website documentation. This guide will walk you through the process of installing, setting up, and managing your hotel booking platform. Whether you’re running a single hotel or a chain of properties, this system provides a user-friendly interface for guests to check availability, book rooms, and make payments online.

Features

  • Responsive design for desktop, tablet, and mobile devices.
  • Interactive room gallery with descriptions and prices.
  • Integrated booking form with date picker.
  • Real-time room availability check.
  • Multiple payment gateway support (PayPal, Stripe, Razorpay).
  • Customizable color schemes and typography via CSS variables.
  • SEO-friendly structure and clean code.
  • Contact form with email integration.

System Requirements

  • Web Server : Apache / Nginx
  • PHP : 7.4 or higher
  • MySQL : 5.7 or higher
  • Browser : Latest version of Chrome, Firefox, Edge, or Safari
  • Internet connection for payment processing

Installation

  1. Download the full project zip from your source.
  2. Extract files into your web server directory (e.g., htdocs for XAMPP).
  3. Import the hotel.sql database file into MySQL.
  4. Update database connection details in config.php.
  5. Ensure the uploads directory has proper write permissions.

Setup Guide

Follow these steps to configure your hotel booking website:

  • Open config.php and set your database username, password, and database name.
  • In assets/css/style.css, modify root variables to change colors and fonts.
  • Add your hotel details in admin/settings.php.
  • Test the booking form to ensure it’s connected to your database.

Rooms Section

The rooms section allows you to display your hotel’s offerings. Use the .room-card class for a consistent style.

  • High-resolution images
  • Price per night
  • Short description
  • Book Now button linking to the booking form

Booking System

The booking system lets guests select check-in and check-out dates, choose rooms, and confirm reservations. It includes :

  • Date picker using Flatpickr.
  • Dynamic price calculation.
  • Email confirmation upon successful booking.
  • Admin panel to manage and approve bookings.

Payment Integration

The platform supports multiple payment gateways. To set up payments:

  1. Open payment/config.php.
  2. Enter your payment gateway API keys.
  3. Enable or disable payment methods from the admin panel.
  4. Test transactions in sandbox mode before going live.

Contact Page

The contact page includes your hotel’s location, phone number, email, and a contact form. The form sends messages directly to your email using PHP’s mail() function or SMTP.

Frequently Asked Questions

  • Can I add more rooms? Yes, use the admin panel to add unlimited rooms.
  • Is the site mobile-friendly? Yes, it is fully responsive.
  • Can I change the currency? Yes, update the currency symbol in config.php.
